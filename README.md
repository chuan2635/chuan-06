# 🏠 樓層平面 AutoCAD 轉換工具

將手繪丈量圖自動轉換成 AutoCAD DXF 檔案的智能工具。

## ✨ 主要功能

- 📷 **自動識別尺寸標註** - 使用 OCR 識別圖上的數字尺寸
- 🎯 **牆面檢測** - 自動檢測黑色線條作為牆面邊界
- 🚪 **門窗識別** - 檢測圓圈和特殊標記識別門窗位置
- 🎨 **顏色分類** - 區分紅色（天花板高度）和綠色（水電冷氣）標註
- 📊 **DXF 輸出** - 生成標準的 AutoCAD DXF 格式檔案
- ⚙️ **靈活設定** - 可調整標度尺、牆厚、門寬等參數
- 🖼️ **實時預覽** - 上傳後即時檢視識別結果

## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 啟動伺服器

```bash
npm start
```

服務器將在 `http://localhost:3000` 啟動

### 使用方法

1. 打開瀏覽器訪問 `http://localhost:3000`
2. 拖拽或選擇你的丈量圖
3. 調整轉換設定（標度尺、單位等）
4. 點擊「開始轉換」進行識別
5. 查看識別結果並調整
6. 點擊「下載 DXF 檔案」

## 📋 輸入要求

### 推薦的丈量圖格式

- **格式**：JPG, PNG（PDF 支援待實現）
- **清晰度**：至少 300dpi 的掃描件或高質量照片
- **尺寸**：A4 或更大的尺寸

### 你的標註習慣

根據你的標註習慣，工具支持：
- 面對那面牆，數字就對著那面牆的標註方式
- **紅色字**：天花板高度和樑的高度
- **綠色字**：水電冷氣套管排水等設施
- **黑色線**：牆面和空間邊界
- **圓圈**：門窗位置

## ⚙️ 轉換參數

### 基本設定

| 參數 | 說明 | 預設值 |
|------|------|--------|
| 標度尺 | 丈量圖中 1 像素相當於實際的 cm | 1 |
| 輸出單位 | DXF 檔案的單位 (cm/m/mm) | cm |
| 預設牆厚 | 牆面的預設厚度 | 15 cm |
| 預設門寬 | 門的預設寬度 | 90 cm |

### 進階設定

| 參數 | 說明 | 預設值 |
|------|------|--------|
| OCR 語言 | 識別文字的語言 | 繁體中文 |
| 邊界檢測靈敏度 | 線條檢測的敏感程度 | 50 |
| 最小線條長度 | 有效線條的最小像素長度 | 20 |
| 自動檢測門窗 | 是否自動檢測門窗位置 | 是 |
| 自動檢測牆面 | 是否自動檢測牆面線條 | 是 |

## 📊 識別結果

處理完成後，工具將顯示：

- **識別的尺寸標註**：找到的所有數字尺寸
- **檢測的牆面**：識別的牆面線條數量和位置
- **檢測的門窗**：找到的門窗位置和數量
- **識別的文字**：按顏色分類的特殊標註

## 💾 輸出檔案

### DXF 檔案

包含以下圖層：
- **WALL** - 牆面線條
- **DOOR** - 門窗圓圈
- **DIMENSIONS** - 尺寸標註

可用以下軟件打開：
- AutoCAD
- LibreCAD
- Inkscape
- 其他 CAD 軟件

### 識別報告 (JSON)

包含所有識別的數據，便於進一步処理或手動調整。

## 🔧 技術棧

- **前端**：HTML5 + CSS3 + JavaScript
- **後端**：Node.js + Express.js
- **圖像處理**：Jimp
- **OCR**：Tesseract.js
- **輸出**：原生 DXF 格式生成器

## 📝 工作流程

```
丈量圖 → 圖像處理 → 線條檢測 → OCR 識別 → 結果預覽 → DXF 生成 → 下載檔案
        ↓                    ↓
        邊界檢測            文字分類
        ↓
        門窗檢測
```

## 🐛 已知限制

- PDF 輸入暫不支援（可先轉換為 PNG/JPG）
- 複雜的手繪圖可能需要手動調整識別結果
- OCR 精度取決於圖像質量
- 目前不支援斜線和複雜曲線自動識別
